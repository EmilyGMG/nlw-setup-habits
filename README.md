This application you can control your routine and your habits.

you need .env file:

`DATABASE_URL="file:./dev.db"`
## Run Locally

```bash
  git clone https://github.com/EmilyGMG/nlw-setup-habits.git
```

```bash
  cd habits
```
```bash
  cd server
  npm install
```

```bash
  cd web
  npm install
```

```bash
  cd mobile
  npm install
```

Start the server and web:

```bash
  npm run dev
```

Start mobile:

```bash
  npx expo start
